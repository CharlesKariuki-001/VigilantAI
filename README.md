### 🛠️ Tech Stack

- **Core**: Python 3.10+
- **Rule Engine**: Custom weighted regex patterns (English + Swahili + Sheng)
- **Machine Learning**: XGBoost + TF-IDF + Structural Features
- **Explainability**: SHAP (TreeExplainer)
- **Frontend**: Streamlit
- **Data**: pandas, CSV-based feedback & reports
- **Others**: joblib, imbalanced-learn

---

### 🎯 Why This Project Matters

Mobile money fraud is a big problem in Kenya and East Africa. Many people lose hard-earned money to sophisticated scams. Vigilant AI aims to give ordinary users (boda riders, market traders, students, mothers) a simple, free, and trustworthy tool to check suspicious messages before they act.

This is not just a school project — it’s a **real attempt** to solve a real problem affecting millions of Kenyans.
