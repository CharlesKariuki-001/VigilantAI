### 🎯 Mission
Vigilant AI detects **social engineering scams**, **SIM swap attempts**, **fake M-Pesa messages**, and anomalous transactions in real time.  

It is purpose-built for the African mobile money context — Swahili urgency language, Sheng slang, Safaricom message patterns, and the realities of users on feature phones and 2G networks.

This is **not** another generic fraud tool. It is a layered system designed from the ground up in Thika, Kenya.

### ✨ Current Features (Phase 1 — Rule Engine)
- ✅ **Rule-based detection** with 30+ regex patterns tailored to Kenyan scams
- ✅ **Streamlit web interface** — simple, fast, works on low-bandwidth
- ✅ **Plain English + Swahili explanations**
- ✅ **Synthetic + real Kenyan scam dataset** generation tools
- 🔄 **Machine Learning Classifier** (in progress — Isolation Forest / Random Forest)
- 🔄 SHAP explainability (coming soon)
